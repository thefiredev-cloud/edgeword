# EdgeWord Landing Page

**Intelligence That Never Sleeps**

Real-time competitive monitoring across every signal — hiring, pricing, reviews, patents, social.

## Tech Stack

- **Next.js 14** (App Router, static export)
- **React 18**
- **Three.js** via @react-three/fiber + @react-three/drei
- **Tailwind CSS**
- **TypeScript**

## Features

- Rotating 3D wireframe globe with connection points
- Glassmorphism navigation
- Fully responsive design
- Static export ready for deployment
- Zero external dependencies for icons/images

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Build

```bash
npm run build
```

Outputs to `out/` directory for static hosting.

## Deployment

Static export ready for:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting service

## Design

- Background: `#0a0a0f`
- Accent: `#10b981` (emerald)
- Text: white / `#94a3b8`
- Glassmorphism effects throughout
